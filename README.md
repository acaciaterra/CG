# CG

#Para compilar:
  ```
  g++ *.cpp -O2 -lGL -lGLU -lglut
  ```
  ou
  ```
  make
  ```

#Para executar:
  ```
  ./a.out
  ```
#Controles:
- Com os botões de 1 a 5 é possível mudar a posição da camêra ao redor da cena.
- O botão '+' possibilita uma visão mais de cima.
- O botão '-' retorna a camêra para a posição normal.
